# Setup wordpress and its monitoring



### Introduction
Ansible script to setup Wordpress and its monitoring

Configuration
~~~~~~~~~~~~~

    $ mkdir workdir
    $ cd workdir
    $ git clone https://github.com/prabhatmaurya/pVagrant.git
    $ cd ch4
    $ vagrant up
~~~~~~~~~~~~~
